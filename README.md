# StudentSelector 添码俱乐部项目——点名系统
## 文件规范
- 本目录应只存储网站本身文件，不包括数据库
- 严禁3级以上目录
## 代码规范
- 缩进！！缩进！！缩进！！！
- 变量和函数名不得中英文拼音混杂，应为100%英文
- coding 100% utf-8
### 变量规范
- 用户提交变量使用"s_"作为开头
- 其他自便
## 版本管理
- 稳定版出现后应推到另一个branch

## 外部库使用
- 使用了 [normalize.css](https://github.com/necolas/normalize.css)
- 使用了 [anijs](https://github.com/anijs/anijs)
